Python Jumble Words Game

This is a small Python project of a two player word guessing game played in the terminal.

In this game a word is chosen and shown in a jumbled (mixed) form.  
Players take turns trying to guess the original word.  
If the guess is correct, the player gets 1 point.  
If the guess is wrong, no point is added.

The game keeps running until players decide to stop.  
At the end, scores are shown and the winner is printed.

How to run

1. Make sure Python 3 is installed
2. Open terminal in the project folder
3. Run:

python gamejumblewords.py


What the program does

- Asks names of two players
- Gives a jumbled word
- Alternates turns between players
- Updates score
- Asks if players want to continue
- Declares winner at the end


Files

gamejumblewords.py - main game program  
hellotrial.py - test file


Purpose

This project was made to practice basic Python concepts like loops, functions, input/output and conditions.
